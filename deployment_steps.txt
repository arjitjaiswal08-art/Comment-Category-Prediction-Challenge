OPTION A: Render.com (RECOMMENDED - EASIEST)
────────────────────────────────────────────────────────────────

□ Create GitHub account (if needed): github.com/signup
□ Create new repository: github.com/new
  - Name: comment-predictor
  - Public
□ In Terminal, run:
  □ git init
  □ git add .
  □ git commit -m "Deploy ML model"
  □ git remote add origin https://github.com/YOUR_USERNAME/comment-predictor.git
  □ git push -u origin main
□ Go to render.com
□ Sign up with GitHub
□ Click "New +" → "Web Service"
□ Select your repository
□ Configure:
  - Name: comment-predictor
  - Environment: Python 3
  - Build: pip install -r requirements.txt
  - Start: gunicorn --bind 0.0.0.0:$PORT --workers 4 --timeout 120 app:app
  - Instance: Free
□ Click "Create Web Service"
□ Wait 5-10 minutes
□ Copy your URL: https://comment-predictor-xxxx.onrender.com
